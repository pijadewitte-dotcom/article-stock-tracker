# Article Stock Tracker

Een statische stock tracker met een Vercel API-route voor Yahoo Finance-data.

[Deploy with Vercel](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fpijadewitte-dotcom%2Farticle-stock-tracker)

## Gebruik

- Plak een ticker of Yahoo Finance-link in het ticker-veld, bijvoorbeeld `AAPL` of `https://finance.yahoo.com/quote/AAPL`.
- Klik op **Prijs** om Yahoo-data op te halen.
- Voeg eventueel artikeltekst of een screenshot toe.
